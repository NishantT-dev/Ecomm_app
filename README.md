#  E-Commerce App Backend (Node.js)
[Created by Nishant Thakur]
##  Features
- User Registration & Login
- JWT Authentication
- Role based Authentication (Admin,seller,user)
- Cart Creation and Checkout 
##  Tech Stack
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- bcryptjs
- express-validator
- Stripe payment integration (basic)
